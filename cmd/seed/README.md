# Seed Data для тестирования

Скрипт для заполнения базы данных тестовыми данными.

## Использование

### Через Makefile (рекомендуется):
```bash
make seed
```

### Напрямую:
```bash
go run cmd/seed/main.go
```

## Примечания

- Все пользователи уже верифицированы (IsVerified = true)
- Пароли хешируются с помощью bcrypt
- Пользователи привязаны к задачам (many-to-many)
- Задачи имеют различные статусы, приоритеты и даты
- Сообщения в чате созданы для нескольких задач

## Очистка данных

Если нужно очистить данные перед seed, раскомментируйте строку в `main.go`:
```go
db.Exec("TRUNCATE TABLE users, projects, tasks, chat_messages CASCADE")
```

**Внимание:** Это удалит все существующие данные!
